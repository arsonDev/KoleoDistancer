# KoleoDistancer
Aplikacja obliczająca dystans między stacjami

## Jak uruchomic
### Sposób 1
- Pobierz Android Studio </br>
- Skompiluj projekt i nacisnij start </br>
### Sposób 2
- Pobierz plik app-debug.apk z zasobów repozytorium i zainstaluj na urządzeniu
